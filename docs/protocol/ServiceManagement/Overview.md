# 服务管理 API

[← 返回协议概览](../Overview.md)

## 概述

服务管理负责服务实例级别的管理操作，包括管理员的增删查改，确保服务的安全性和可维护性。

## API 列表

### 📖 [获取管理员](./GetAdmin.md)
获取服务实例的所有管理员列表

### ✏️ [增加管理员](./AddAdmin.md)
设置某一个或多个用户为管理员

### ✏️ [撤销管理员](./RevokeAdmin.md)
撤销指定用户的管理员权限

## 权限要求

所有服务管理 API 都需要调用者是现有管理员，确保服务管理的安全性。

## 返回格式

所有 API 都遵循统一的返回格式：
- 成功时返回 HTTP 200 和对应的 Result 类型
- 失败时返回对应的 HTTP 错误状态码和 Error 类型
